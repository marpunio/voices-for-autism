Create your own Alexa commands with Voices for Autism

Voices for Autism is a program that allows parents to write out commands for their children on any end of the autism spectrum. The command will them be automated to an Amazon Echo Dot, where Alexa will be able to spit out the command to your child. Here it is! Try Voices for Autism now! Simply type out a command and listen to the Alexa skill get processed into the webpage which will be automated to the Alexa skillskit database for your use! Every voice matters!
